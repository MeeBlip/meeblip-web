MeeBlip Apps

Create and share patches for Anode and Triode using Web MIDI editors for Chrome and Opera.

Quick Start Guide

  1. Click one of the buttons above to open up the editor for your MeeBlip synth
  2. Select the MIDI port and channel your MeeBlip is set up to use from the control bar:
  3. Adjust knobs and switches - you can control MIDI-only params, too! (Use the button to generate a random config or load up a sample patch from the patch browser via the button)
  4. Click the button to transmit knob and switch values to your MeeBlip. (Make sure you have the port and channel settings correct!)
  5. To save a patch, click the button then click the link in the dialog window to load the URL in your browser for bookmarking.
  6. Don't have a keyboard or DAW handy? No problem! Click the button to launch an 8-step sequencer and try out your sounds right from the editor. (Tip: the sequencer can continue playing when its dialog is closed. So you can load up new patches while the sequencer plays in the background.)
  7. Share patch links on Twitter at @MeeblipPatches! Tag with #meeblip and a #your-patch-name tag to help other users find and try your patches.
  8. Code is open source and available on GitHub.
